# Enerflo.ai Marketing Website

Premium curated landing website for Enerflo.ai CRM for UK energy brokers.

## Stack

- Next.js App Router
- React
- Plain CSS with design tokens
- Lucide React icons
- Google Fonts: Inter and Caveat
- Deploy: Vercel

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Pages included

- `/` homepage
- `/features`
- `/pricing`
- `/use-cases`
- `/about`
- `/help`
- `/book-demo`

## Notes

- No login link is included in the marketing navigation.
- The hero contains a feature-video placeholder. Replace it later with your actual 30-60 second promotional feature video.
- Pricing is built around Starter, Growth and Scale.
